modeling.modeling_lightgbm(X_train: DataFrame, y_train: DataFrame, params: dict)[ソース]

lightgbmモデルを使用して予測モデルを作成する関数.

評価指標は二乗平均兵香港誤差(RMSE)を使用している

パラメータ

  • X_train (pd.DataFrame) -- 説明変数データ

  • y_train (pd.DataFrame) -- 目的変数データ

  • params (dict) -- 学習に使用するパラメータ

戻り値

学習の結果

  • X_train: 説明変数データ

  • y_train: 目的変数データ

  • model: 説明変数データ

  • lgb_results: 学習ログ

  • loss_train: 学習ロス

戻り値の型

dict

modeling.modeling_lightgbm_with_cv(X: DataFrame, y: DataFrame, params: dict)[ソース]

クロスバリデーションを使用したlightgbmでの学習を行う.

パラメータ

  • X (pd.DataFrame) -- 説明変数データ

  • y (pd.DataFrame) -- 目的変数データ

  • params (dict) -- 学習パラメータ

戻り値

_description_

  • params: 学習に使用したパラメータ

  • cv_score: cvスコア。各学習で算出したスコアの平均値

  • models: 各学習で作成したモデルのリスト

戻り値の型

dict
